# 🏥 Clinic Management System - Frontend

واجهة أمامية حديثة لنظام إدارة العيادات مبنية بـ React + Vite + TailwindCSS.

## 🚀 الميزات

- 🔐 نظام مصادقة كامل (JWT)
- 📱 تصميم متجاوب يعمل على جميع الأجهزة
- 🎨 واجهة عربية (RTL) مع TailwindCSS
- ⚡ أداء عالي بـ Vite
- 🔗 اتصال تلقائي مع Backend API

## 📦 التقنيات المستخدمة

| التقنية | الإصدار | الغرض |
|---------|---------|-------|
| React | 18.2 | واجهة المستخدم |
| Vite | 5.0 | أداة البناء |
| TailwindCSS | 3.3 | التنسيقات |
| React Router | 6.20 | التوجيه |
| Axios | 1.6 | طلبات HTTP |
| Lucide React | 0.294 | الأيقونات |

## 🛠️ التثبيت المحلي

```bash
# تثبيت المكتبات
npm install

# تشغيل وضع التطوير
npm run dev

# بناء النسخة الإنتاجية
npm run build

# معاينة النسخة الإنتاجية
npm run preview
